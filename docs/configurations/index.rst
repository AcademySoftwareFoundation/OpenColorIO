Configurations
==============

This section gives an overview of what existing (public) OCIO configurations exist,
and how to create new ones.

OCIO Configurations can be downloaded here: `.zip <http://github.com/imageworks/OpenColorIO-Configs/zipball/master>`__ `.tar.gz <http://github.com/imageworks/OpenColorIO-Configs/tarball/master>`__ (OCIO v1.0+)

OCIO Configurations for ACES 1.1 and 1.2 can be downloaded from the `Colour Science <https://github.com/colour-science/OpenColorIO-Configs>`_ repository maintained by Thomas Mansencal here:

- ACES 1.1 (OCIO v1.0+):
    - `OpenColorIO-Configs-1.1.zip <https://github.com/colour-science/OpenColorIO-Configs/archive/v1.1.zip>`_
    - `OpenColorIO-Configs-1.1.tar.gz <https://github.com/colour-science/OpenColorIO-Configs/archive/v1.1.tar.gz>`_
- ACES 1.2 (OCIO v1.0+):
    - `OpenColorIO-Configs-1.2.zip <https://github.com/colour-science/OpenColorIO-Configs/archive/v1.2.zip>`_
    - `OpenColorIO-Configs-1.2.tar.gz <https://github.com/colour-science/OpenColorIO-Configs/archive/v1.2.tar.gz>`_

If you are interested in crafting custom color configurations, and need assistance, please contact: `ocio-user <https://lists.aswf.io/g/ocio-user>`__\.
