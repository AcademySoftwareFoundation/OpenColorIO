For Artists

OCIO is integrated and ready to use in many major software packages - check out Supported Applications for more information.

For Developers

Brand new to color management frameworks? Upgrading to OCIO v2? Our Software Developer Guide has all the details.

OCIO v2 is here!

Check out our guides for upgrading. Here are just a few highlights of new features:

New GPU Renderer

The GPU renderer now matches the CPU and may be used to render final frames!

Better User Experience

Config authors have new tools to organize color spaces by category, hide color spaces, and define hierarchical menus. New File and Viewing Rules allow more powerful default behaviors.

Improved ACES Support

ACES Output Transforms no longer use 3d-LUTs and are accurate even for extreme exposure values. There is also full read and write support for the Academy/ASC Common LUT Format (CLF).

Analysis Tools

There are new command line tools for evaluating LUTs, converting to CLF, serializing OCIO Processors, and more!
